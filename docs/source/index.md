## 关于创极语                     

创极语是一种与眼、口、耳有关的、属于优选数值型、优选声节型概念标记号的人类通用语言文字。

### 为什么叫“创极语”？

西文名称 Colating General Human Language 一直没有变，变得只是中文译名。   
- “创极语”：致力于为人类无限创造能力服务的一种极限型人类通用语文；简称：Colating Language。  
  - 创：关注人类先天所具有的无限智慧能力和创造能力的新型优选数值和优选声节语文，属于一种下一代的拉丁语。  
  - 极：极限，试图匹配人类智慧的相对无限创造能力作为本语言的服务目标。
- 科极语：科学的极限型人类通用语文，“科学”这个概念的目的性过于模糊了。
- 科䁒语：科学的、与眼、口、耳有关的人类通用语言。什么是“䁒”？表明这不是盲文，也不是哑语或其他；但也不想使用这个单字，以免与历史释义纠缠不清。
- 还有其他的一些译名。
- Colating Language在创造的过程中，曾经为了概念总量可能难以满足长度数千年的需求而停滞以便进一步进行智慧思考。
  - 当将高效概念标记号总量提升到 9999年的需求，甚至次高效的概念重量可以提升到数万年，甚至是数百万年时，又再次停滞了，因为这也是非常有问题的。
  - 新车代谢，变是永远的不变，这才是宇宙的客观事实。后来强行设置为每 4K年左右必须轮换一次，将 9001~9999年强行设置为其他用途，以避免被滥用。

### 在 Colating设计中需要考量的一些规则：
- 语言学习时长：个体平均寿命为 70年，掐头去尾，能够用于创造的时间可能就并不太多了。因此，怎样将学习 Colating通用语文的时间压缩到极限呢？Colating期望参数设计为：
  - 婴幼儿母语 5年；
  - 成人母语 1~2年；
  - 非母语成人 2~3年。
- 概念总量：人类先天的智慧能力是无限的，创造能力虽然是小于智慧能力，但也是相对无限。因此，怎样将原子概念标识号的总量提升到人类语言能力的极限呢？Colating期望参数设计为：逼近正常人类语文能力的极限；目前估计为英文 5倍以上；这主要是受限于人类有限的发音能力造成的。未来通过调整字母的发音，从而可以获得更大的可用概念标记号总量。
- 概念标记号总量与英文相比，这个 5倍是一个非常保守的估计，实际上可能还要大很多，并且 colating 的概念标记号的总量还具有良好的可扩充性。
- 显式局部标记功能字母：即使 Colating目前有效的概念总量是英文的 5倍，甚至是更多倍，它与宇宙中概念对象的总量相比，由于受限于人类天生过于局限的发音能力，因此概念标记号的总量还是太少了，就算概念标记号的总量增加上再多，也是会枯竭的。Colating通过引入显式局部标记功能字母来解决了这个难题，类似于编程语言中的局部变量，显式标记避免了语言场景的歧义问题。
- Colating标准构词法，使得在更大的概念标记号总量的场景里，依然可以保持清晰的构词法规则。这是现行拉丁语所难以办到的，因此说：Colating 属于下一代的拉丁语。
- Colating扩展构词法：概念标记号是用来标记概念对象的。同时存在有规则与不规则的概念对象，这是宇宙的客观现实。因此，Colating同样存在有非规则构词法，但这类概念标记号的数量是可控的，或者是可识别的。
- 通过 Colating扩展构词法，可以创造出高效的、可用的、数量庞大的原子概念标记号。
- 新城代谢，轮转机制：必须限制 Colating的轮换时长，必须具备有自我代谢的能力。就像地球上的动物一样，只吃不拉的貔貅，很快就绝迹了。
- 单词的发音：虽然情绪激发家（小说、故事，歌曲等等）可以激发出人类正面的、或负面的情绪，这对人类未来进入宇宙社会并没有什么直接的帮助；但“望梅止渴”也是必须的一种存在。Colating保留了专门一个全域的、可以满足歌曲家、诗歌家等创造的原子概念标记号集合；在这个集合中的概念标记号，忠实记录自各种情绪激发类的创作作品。现行拉丁语的发音，是这个概念标记号专门集合发音的子集；但通过语文词典规则，将这个概念标记号专门集合限制并作为现行拉丁语发音的一个很小的集合来使用，以避免不规则概念标记号的泛滥。
- 单纯 AI机器、或 AI人类创造的概念标记号与自然人类所创造的概念标记号，可以无缝衔接。AI人类（未来很可能会出现：AI机器与人类部件相结合的产物，当前的脑机接口已趋于成熟，因此，这可能并不是在做梦。）
- 规避了在其他拉丁语中的一些已知的 BUG。
- Colating标准构词法的格式：`前缀+前缀功能字母++前音缀+前音缀功能字母 ++词根（包含根缀标记功能字母）++后音缀+后音缀功能字母++后缀功能字母+后缀`，在一个实际的单词中，一般只有其中的部分功能字母。
- 需要创造什么样的概念标记号，才能满足如此庞大、且持续增长的概念对象总量的标记呢？
- 概念标记号总量与使用该标记号的人口总量、创造总量等有关。
- 概念标记号失效的评价规则和失效机理。 
- Colating扩展构词法的格式：待发布。。。。。。
  
Colating标准构词法，单词最短为 3个字母。虽然总量在 >2K (?)词时图形标记号的表达效率略低于现代拉丁语；但通过扩展构词法泥补了这部分的不足。

## colating language 构词法、语法

- 特点    
  - 有效的原子概念标记号的创造，永远掌握在概念创造者的手中。
  - 创极语概念标记号（相当于：英语单词）的理解与传播，可以不需要第三者的介入。
  - 易于理解，易于创造原子概念标记号。
  - 兼容在未来 AI 可能可以自行创造的概念标记号。
  - 创极语的创造，以无限语言哲学理论作为基础支撑。

### 创极语的权利要求

最小约束条件，为了保证 Colating Language 创造的顺利进行， Colating 字母采用 类似软件行业的 MIT许可，默认有 4种常用拉丁字重（Light‌，‌Regular、‌Bold 和 ‌Italic）的权利要求。

- Colating 人类通用语言的许可，不会高于、或多于任何一种联合国官方语言。但也不会超出不同于联合国官方语言中的不合理部分。  
- Colating 为了确保本语言文字能够被正确使用，会发布无版权要求的语法（含构词法）等的资料。  
- Colango (clgo-) 的理论创作作品，有版权要求,此收入作为 Colating开发的费用之一，未来期望有基金会进行管理。目前其他主流语言的思想作品，也是有版权要求的。
- 通用语言本身是没有版权要求的，而思想作品是有版权要求的，注意它们之间的区别：拉丁语语言本身是没有版权，任何人的可以使用它；而采用拉丁语写了一部流行小说，这部小说是有版权要求的。
- 其他，待完善。
